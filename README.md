# ASCII Art Installation

Transform images into interactive, living ASCII art.

## Features

- **Image to ASCII conversion** - Upload any image and watch it transform into dynamic text art
- **Interactive effects** - Wind, Ripple, Magnetic, Glitch, Smear, and Chaos modes
- **Character sets** - Toggle between Unicode blocks and ASCII letters
- **Density modes** - Standard, HD, and Minimal for different performance/quality tradeoffs
- **Export** - Save as PNG (2x resolution) or plain text
- **Gallery** - Load multiple images and navigate with wave reveal transitions
- **Mobile support** - Touch gestures and optimized performance

## Usage

1. Click "New" to upload an image
2. Interact with the canvas - hover, click, drag
3. Use the mode selector at the bottom to change effects
4. Toggle Sound, Color, Drift, Character set, and Density
5. Navigate between images with arrow keys or swipe
6. Click "Save" to export as PNG or text

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- Vanilla JavaScript
- HTML Canvas
- Vite (build tool)
- Web Audio API (sound effects)
